### Endpoints 
- All Pokémon: https://pokeapi.co/api/v2/pokemon?limit={name}
- Just a Pokémon: https://pokeapi.co/api/v2/pokemon/{id}/{name}
- Generations: https://pokeapi.co/api/v2/generation/{number}
